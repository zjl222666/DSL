概述
------------
SerWise 是为自动回复服务程序设计的 **领域特定语言(domain specific language)** .

不同与传统的高级语言，如C++,python等，Serwice语言不提供数据结构和面向对象编程具有极强针对性，纯粹的过程式编程和自由的语法规则能够适应绝大多数场景需求，而对于特殊场景，解释器给出了充分的可修改，可控制，可添加的API，给脚本编写者提供了极大的发挥空间

考虑自动应答服务与NLP，TTS等技术的紧密联系性，解释器底层使用 **python编写**，并对算法工程师和维护人员开放API，可以实时根据需求修改内部执行命令
